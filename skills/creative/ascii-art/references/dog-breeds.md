# Dog Breed ASCII Art Reference

## Australian Shepherd — Black Tri, Docked Tail

```
           __      _
          /  \    / \
         / ..|\  /| .. \
        (  >< | \/ | ><  )   <-- Brown eyes
         \ __ |    | __ /
          \  \|    |/  /
           `""""""""`     
            |      |          <-- Black saddle
         ___|      |___
        /  \________/  \
       /   /        \   \
      /   /          \   \
     |   |            |   |
     |   |            |   |   <-- Tan points on cheeks/legs
     |   |            |   |   <-- White blaze, chest, legs
      \   \          /   /
       \   \________/   /
        \______________/
         |  |    |  |
         |  |    |  |   <-- Docked tail (nub)
         |  |    |  |
        _|__|____|__|_
       (______________)
```

Key features: Black saddle/back, tan (copper) points on cheeks & legs, white blaze up face + chest + legs, brown eyes, **docked tail (short nub)**. Slightly larger/blockier than Border Collie.

## Border Collie — Black & White, Heterochromia

```
          __      _
         /  \    / \
        / ..|\  /| .. \
       (  @ | \/ | O  )   <-- Blue eye (@) + Brown eye (O)
        \ __ |    | __ /
         \  \|    |/  /
          `""""""""`     
           |    |           <-- White blaze
        ___|    |___
       /  \________/  \
      /   /        \   \
     /   /          \   \
    |   |            |   |
    |   |            |   |   <-- Black patches on white
    |   |            |   |   <-- Classic "Irish spotting" pattern
     \   \          /   /
      \   \________/   /
       \______________/
        |  |    |  |
        |  |    |  |   <-- Full fluffy tail
        |  |    |  |
       _|__|____|__|_
      (______________)
```

Key features: Black & white only (no tan), white blaze up face, black patches on body, **heterochromia** (one blue, one brown eye), **full long tail**, slightly smaller/more athletic build than Aussie.

## Technique Tips for Recognizable Dog ASCII

1. **Head shape matters** — Use `/ \` for ears, `( )` for face, distinguish ear types (floppy vs semi-erect)
2. **Eyes are critical** — Different chars for colors: `@`=blue, `O`=brown, `●`=dark, `><`=amber
3. **Snout** — `\ __ /` or similar gives depth
4. **Body proportions** — Dogs are longer than tall; legs at corners
5. **Tail tells the breed** — Docked (nub) vs full plume vs curl
6. **Markings via comments** — Since ASCII is monochrome, use inline comments to indicate color zones
7. **Reference real silhouettes** — Sketch the outline first, add details second
8. **Avoid abstract blobs** — The user rejected "░░░" pattern fills; they want anatomical shapes
