# MSYS2 / Git-Bash Shell Setup

> Shell config for the AI terminal stack on native Windows (MSYS2 git-bash).
> Created 2026-06-19 from session where fzf crashed without fd, and zellij layout
> needed absolute paths.

## Files

| File | Purpose |
|------|---------|
| `~/.bashrc` | Interactive shell config: aliases, fzf, fd, prompt |
| `~/.bash_profile` | Login shell: sources `.bashrc` and `.profile` (auto-generated by Git for Windows) |
| `~/.fzf.key-bindings.bash` | Downloaded from fzf repo — Ctrl-T, Ctrl-R, Alt-C bindings |
| `~/.fzf.completion.bash` | Downloaded from fzf repo — fuzzy completion for commands |

## .bashrc Structure

### 1. PATH Additions

```bash
# Bun (Hermes CLI)
export PATH="$HOME/.bun/bin:$PATH"

# MinGW GCC (for tree-sitter parser compilation)
MINGW_DIR="/c/Users/<user>/AppData/Local/Microsoft/WinGet/Packages/\
  BrechtSanders.WinLibs.POSIX.UCRT_.../mingw64/bin"
if [ -d "$MINGW_DIR" ]; then
  export PATH="$MINGW_DIR:$PATH"
fi

# fd (fast find — avoids fzf/MSYS2 "find" slowness)
FD_DIR="/c/Users/<user>/AppData/Local/Microsoft/WinGet/Packages/\
  sharkdp.fd_.../fd-v<x.y.z>-x86_64-pc-windows-msvc"
if [ -d "$FD_DIR" ]; then
  export PATH="$FD_DIR:$PATH"
fi
```

### 2. FZF Defaults (set BEFORE sourcing fzf bindings)

```bash
export FZF_DEFAULT_COMMAND="fd --type f --hidden --follow --exclude .git"
export FZF_CTRL_T_COMMAND="$FZF_DEFAULT_COMMAND"
export FZF_ALT_C_COMMAND="fd --type d --hidden --follow --exclude .git"
```

**Why this matters:** On MSYS2, `find` traverses the entire C: drive (minutes).
`fd` is instant. Without this, sourcing the fzf key-bindings script hangs for minutes.

### 5. Custom Functions

```bash
# Quick reference — prints all aliases and shortcuts
cheatsheet() {
  echo ""
  echo "  ╔══════════════════════════════════════════╗"
  echo "  ║   AI Terminal Stack — Quick Reference     ║"
  echo "  ╠══════════════════════════════════════════╣"
  echo "  ║  zellij --layout dev   Start dev layout  ║"
  echo "  ║  z                    Launch zellij       ║"
  echo "  ║  zd                   Dev layout shortcut ║"
  echo "  ║  herm                 Hermes TUI          ║"
  echo "  ║  v  /  vi             Neovim              ║"
  echo "  ║  lg                   lazygit             ║"
  echo "  ║  y                    yazi (file mgr)     ║"
  echo "  ║  Ctrl-T               fzf: files          ║"
  echo "  ║  Ctrl-R               fzf: history        ║"
  echo "  ║  Alt-C                fzf: cd             ║"
  echo "  ║  fg <term>            fzf: grep & open    ║"
  echo "  ╚══════════════════════════════════════════╝"
  echo ""
}

# fzf git log browser — select commit, SHA copied to clipboard
fgl() {
  git log --oneline --graph --color=always | \
    fzf --ansi --no-sort --reverse --tiebreak=index | \
    awk '{print $1}' | head -1 | tr -d '\n' | clip.exe && \
    echo "SHA copied to clipboard"
}

# fuzzy grep — search file contents, open in nvim on Enter
fg() {
  local file
  file=$(rg --ignore-case --files-with-matches "$1" | \
    fzf --preview "bat --color=always --line-range=:100 {}" \
        --preview-window=right:60%)
  [ -n "$file" ] && nvim "$file"
}
```

### 6. PS1 (colored prompt with git branch)

```bash
__git_branch() {
  local branch
  branch=$(git rev-parse --abbrev-ref HEAD 2>/dev/null)
  [ -n "$branch" ] && echo " ($branch)"
}

# Full color prompt in terminal, plain inside neovim terminals
if [ "$TERM" != "dumb" ] && [ -z "$NVIM" ]; then
  PS1='\[\e[1;32m\]\u\[\e[m\] \[\e[1;34m\]\w\[\e[m\]\[\e[1;33m\]$(__git_branch)\[\e[m\] \$ '
else
  PS1='\w \$ '
fi
```

## FZF Key Bindings (from downloaded scripts)

| Shortcut | Action |
|----------|--------|
| `Ctrl-T` | Fuzzy file search (from current dir, using `fd`) |
| `Ctrl-R` | Fuzzy command history search |
| `Alt-C` | Fuzzy directory cd (using `fd --type d`) |

## Troubleshooting

### "source .bashrc hangs for minutes"

**Cause:** fzf key-bindings trigger `find` which traverses C: drive on MSYS2.
**Fix:** Install `fd` via winget, set `FZF_DEFAULT_COMMAND` to use `fd` BEFORE
sourcing the key-bindings. See section 2 above.

### "fzf key bindings don't work"

**Cause:** winget fzf package ships only `fzf.exe`, not shell integration scripts.
**Fix:** Download `key-bindings.bash` and `completion.bash` from fzf GitHub repo.
See SKILL.md for curl commands.

### "gcc: command not found in new shell"

**Cause:** Winget adds MinGW to User PATH, but the shell was started before install.
**Fix:** Close and reopen the terminal, or reconstruct PATH from registry:
```powershell
$env:Path = [Environment]::GetEnvironmentVariable("Path", "Machine") + ";" +
            [Environment]::GetEnvironmentVariable("Path", "User")
```
